## Coverage invariant (why registry+defaults+regen is ONE atomic bead)

`validate_sources()` (`codegen/generate.py:200`) runs UNCONDITIONALLY from
`main()` at `generate.py:1281` — it fires even under `--check`. For every object
it enforces (generate.py:307-315):

- `covered = objectDefaults ∪ requiredProperties` must EQUAL the object's full
  registry property set (no missing, no extra), and
- `objectDefaults ∩ requiredProperties` must be EMPTY (a property is defaulted
  XOR required, never both).

For transformConstraint's 8 properties:
`{name, bone, target}` (required) ∪ `{order, translateMix, rotateMix, scaleMix,
shearMix}` (defaulted) = all 8, disjoint. ✓

Because the object-membership list, the defaults, and requiredProperties are
cross-checked in one pass, the registry edit + defaults edit + codegen regen
CANNOT be split — any half fails `validate_sources()` alone. Hence bony-8i1.2 is
atomic.
